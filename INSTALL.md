# oh-my-pdf 설치하기 (터미널 필요 없음)

이 앱은 Apple에 등록되지 않은 무료 앱이라, 처음 한 번만 macOS가 "확인되지 않은 앱"이라며 막을 수 있습니다.
아래 순서대로 설치하면 됩니다.

> 필요 환경: macOS 26 이상

## 1단계 · 내려받아 설치

1. [Releases 페이지](../../releases/latest) 에서 `oh-my-pdf-x.x.x.dmg` 를 내려받습니다.
2. 내려받은 `.dmg` 파일을 더블클릭합니다.
3. 창이 열리면 `oh-my-pdf` 아이콘을 옆의 `Applications` 폴더로 드래그합니다.

```text
┌─────────────────────────────┐
│                             │
│   oh-my-pdf.app   ─────▶   Applications
│                             │
└─────────────────────────────┘
```

4. 설치 끝. 창은 닫고 `.dmg` 파일은 버려도 됩니다.

## 2단계 · 처음 한 번 "그래도 열기"

설치 직후 앱을 처음 열면 이런 경고가 뜰 수 있습니다.

> "oh-my-pdf을(를) 열 수 없습니다. Apple에서 악성 소프트웨어가 있는지 확인할 수 없기 때문입니다."

이때 아래처럼 하세요.

1. 경고창의 `완료` 또는 `취소`를 눌러 닫습니다.
2. 화면 왼쪽 위 Apple 메뉴 → 시스템 설정을 엽니다.
3. `개인정보 보호 및 보안`으로 갑니다.
4. 아래쪽 `보안` 항목에서 `oh-my-pdf이(가) 차단되었습니다...` 문구를 찾습니다.
5. `그래도 열기`를 누릅니다.
6. 한 번 더 물어보면 `그래도 열기`를 누르고 비밀번호 또는 Touch ID로 확인합니다.

이 과정은 설치 후 처음 한 번만 하면 됩니다.

## OpenAI API 키와 PDF 텍스트

맞춤법 검사를 실행하면 PDF에서 추출한 텍스트가 사용자가 입력한 OpenAI API 키로 OpenAI API에 전송됩니다.
API 키는 macOS Keychain에 저장됩니다.

## 업데이트

앱은 Sparkle을 사용해 자동 업데이트를 확인합니다. 새 버전이 나오면 앱 안에서 업데이트를 설치할 수 있습니다.

직접 확인하려면 앱 메뉴에서 `업데이트 확인...`을 누르세요.

## 잘 안 될 때

**"손상되었거나 휴지통으로 옮겨야 한다"고 나와요**

다운로드가 깨졌을 수 있습니다. `.dmg`를 지우고 Releases에서 다시 받아 보세요.

**`그래도 열기` 버튼이 안 보여요**

앱을 한 번 더블클릭해서 열기 시도를 해야 버튼이 나타납니다.

## 터미널을 쓸 수 있다면

보안 차단 해제를 터미널에서 처리할 수도 있습니다.

```bash
xattr -dr com.apple.quarantine /Applications/oh-my-pdf.app
```
