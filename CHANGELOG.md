# 변경 기록

## 0.1.6

- 텍스트 박스와 화살표 메모를 클릭한 PDF 위치에 생성하도록 개선
- 텍스트 박스 생성 직후 PDF 위에서 바로 인라인 편집하도록 개선
- 텍스트 박스 더블클릭/수정 시 모달 대신 PDF 위 인라인 편집 사용
- 텍스트 박스 생성 후 자동으로 선택 모드로 돌아가도록 개선
- 비어 있는 텍스트 박스는 편집 종료 시 자동 제거

## 0.1.5

- 주석 목록을 Acrobat Reader 댓글 패널처럼 카드형 행으로 개선
- 선택한 주석의 메모를 왼쪽 패널에서 바로 수정하고 저장할 수 있도록 개선
- 선택된 주석 행의 강조 표시와 가독성 개선

## 0.1.4

- 왼쪽 주석 목록 추가: 모든 주석을 페이지 번호와 함께 표시
- 주석 클릭/목록 선택 후 수정 및 삭제 지원
- 주석 도구 버튼을 토글 상태로 표시
- AI 맞춤법 교정 결과를 별도 결과 패널 없이 `[AI] 원문 -> 교정문` 하이라이트 메모로 표시
- Command+Z와 Control+Z 되돌리기, Delete 삭제, Return 수정 단축키 지원

## 0.1.3

- PDF를 열었을 때 빈 화면처럼 보이던 레이아웃 문제 수정
- PDF 뷰어를 단일 페이지 맞춤 보기로 변경
- AI 교정 패널이 항상 오른쪽에 보이도록 레이아웃 고정

## 0.1.2

- DMG 설치 안내 문서 추가
- `oh-my-pdf` 고정 자체서명 인증서 생성 스크립트 추가
- 릴리스 빌드가 고정 인증서로 재서명되도록 배포 스크립트 정리

## 0.1.1

- Sparkle appcast와 업데이트 ZIP 게시
- 자동 업데이트 서명 계정을 `oh-my-pdf`로 고정

## 0.1.0

- PDFKit 기반 PDF 열기, 보기, 페이지 이동, 저장 기능 추가
- 하이라이트, 밑줄, 취소선, 텍스트 박스, 화살표 메모, 선택 텍스트 주석 추가
- CSV before/after 용어집과 OpenAI GPT 교정 파이프라인 추가
- Sparkle 자동 업데이트 공개키 설정
